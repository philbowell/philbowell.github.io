<header class="colophon-hero">
  <p class="colophon-kicker">About this website</p>
  <h1>Colophon</h1>
  <p>This site is designed and maintained by Phil Bowell as a small, practical home for portfolio notes, professional history, and future writing.</p>
</header>

<section class="colophon-section" aria-labelledby="design">
  <h2 id="design">Design</h2>
  <p>The visual direction is intentionally quiet: warm off-white background, near-black text, muted grey supporting copy, and a sage green accent used sparingly for links and active details.</p>
  <p>The layout favours generous spacing, narrow measures, and a simple content hierarchy so the writing and work can carry the page without extra decoration.</p>
</section>

<section class="colophon-section" aria-labelledby="type">
  <h2 id="type">Type</h2>
  <p>The site is set in Figtree, served through Google Fonts. It is used across headings, body copy, navigation, and small interface details to keep the system deliberately simple.</p>
</section>

<section class="colophon-section" aria-labelledby="build">
  <h2 id="build">Build</h2>
  <div class="colophon-details">
    <div>
      <p class="colophon-label">Generator</p>
      <p>Jekyll, using the native GitHub Pages build.</p>
    </div>
    <div>
      <p class="colophon-label">Templates</p>
      <p>A single base layout with small includes for navigation and footer.</p>
    </div>
    <div>
      <p class="colophon-label">Styles</p>
      <p>Hand-written CSS in one file, built around custom properties for colour, spacing, and layout.</p>
    </div>
    <div>
      <p class="colophon-label">JavaScript</p>
      <p>None in the initial build.</p>
    </div>
  </div>
</section>

<section class="colophon-section" aria-labelledby="hosting">
  <h2 id="hosting">Hosting</h2>
  <p>The site is hosted on GitHub Pages. Publishing is handled by pushing to the main branch, where GitHub builds and deploys the site automatically.</p>
</section>

<section class="colophon-section" aria-labelledby="materials">
  <h2 id="materials">Materials</h2>
  <p>The wordmark is Phil Bowell Design's own SVG logo. The current content is written directly in Markdown and HTML, with the CV adapted from Phil's existing professional history.</p>
  <p class="colophon-note">Last updated during the 2026 rebuild.</p>
</section>
