# Improve CV role spacing and timeline

Issue: #7

## Goal

Make the Professional history entries on the CV page feel more distinct, with clearer spacing between each role and a subtle vertical timeline with anchor points aligned to each date.

## Changes

- Increase separation between `.cv-role` entries.
- Add a vertical timeline rule inside `.cv-timeline`.
- Add date anchor points for each `.cv-role`.
- Keep the layout responsive on narrow screens.

## Verification

- Inspect the CV page locally in desktop and mobile widths.
- Confirm role content remains readable and dates align with the timeline markers.
